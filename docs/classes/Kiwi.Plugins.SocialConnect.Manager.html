public

Defined in src\manager.js:47

A useful method that allows you to initialise multiple social media SDKS at the same time.

To use this method you must pass it an Object, which contains the configuration settings for each SDK you want to use.

E.g. To initialise Facebook you would pass { facebook: { appId: '1726287530930783' } } Note: appId used for example purposes only.

Parameters:

  • config Object
    • [facebook] Object optional

      Params to init the Facebook SDK. See the 'init' method Facebook for more details.
